## 金融服务销售核心原则

### 1. 建立信任优先于推销
- 先提供价值（免费分析、市场洞察）
- 不要急于推产品，先成为用户的"参考资源"
- 分享真实数据和案例，避免夸大承诺

### 2. 教育式销售（BroadFSC 主要策略）
- 内容：技术分析教程、市场解读
- 互动：回答问题、评论互动
- 转化：在内容末尾自然引导注册/咨询

### 3. 社会认同（Social Proof）
- 分享用户成功故事（需合规，不承诺收益）
- 展示平台功能和优势
- 突出持牌合规的信任背书

