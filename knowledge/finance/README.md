## 核心知识点（手动整理）

### 技术分析基础
- **支撑位/阻力位**: 价格反复触碰但未突破的关键价格区域
- **移动平均线(MA)**: 5日均线 < 20日均线 < 60日均线 趋势判断
- **RSI**: 超过70超买信号，低于30超卖信号，背离形态预示反转
- **MACD**: 金叉（DIFF上穿DEA）买入信号，死叉卖出信号
- **布林带**: 价格触碰上轨超买，触碰下轨超卖，带宽收窄预示突破

### 常见图形形态
- **头肩顶/底**: 趋势反转信号
- **双顶/双底(M顶/W底)**: 强力反转形态
- **三角形整理**: 对称三角突破方向决定下一波方向
- **旗形/楔形**: 趋势延续整理形态

### 蜡烛图形态
- **锤子线**: 下跌末端看涨反转信号
- **吞没形态**: 大阳线/大阴线吞没前一根，强力反转
- **十字星**: 多空平衡，趋势可能转变
- **长上影线/下影线**: 上方/下方遭遇阻力

### 基本面关键指标
- **P/E Ratio**: 市盈率，越低越便宜（对比同行业）
- **EPS**: 每股收益，持续增长是好信号
- **Revenue Growth**: 营收增长率
- **ROE**: 净资产收益率，巴菲特最看重的指标之一

### 宏观经济指标
- **CPI**: 通胀指标，过高→加息预期→利空股市
- **NFP**: 非农就业，超预期好→美股上涨
- **GDP**: 季度数据，衡量经济整体健康
- **Fed Rate Decision**: 美联储利率决议，最重要的宏观事件
