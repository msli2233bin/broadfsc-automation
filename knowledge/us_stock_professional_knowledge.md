## 一、基本面分析 5 步法

### Step 1: 读财报
从10-K开始（年报经审计，投资者演示文稿只是营销）。阅读顺序：利润表→资产负债表→现金流量表→附注。附注里藏着管理层不想让你看的东西。

重点关注：
- 收入增长率（加速还是减速？）
- 毛利率趋势（扩张=定价权强；压缩=竞争激烈）
- 经营现金流 vs 净利润（持续背离是危险信号）
- 债务到期安排（关注利息覆盖率）
- 关联交易（附注中查找表外义务）

### Step 2: 评估盈利质量
三大维度：
1. **应计比率** = (净利润 - 经营现金流) / 总资产 → 越低越好
2. **现金流覆盖率** = 经营现金流 / 净利润 ≥ 1.0x → 应持续高于1.0倍
3. **收入质量** → 增长是有机增长还是并购驱动？是否存在渠道填充？

盈利质量红旗：
- GAAP调整后盈利差距超过20%
- DSO（应收账款周转天数）上升
- 频繁重复的"一次性"费用

### Step 3: 估值（双轨验证法）
必须使用至少两种独立方法：

**方法一：DCF折现现金流**
- 基于未来现金流假设计算绝对内在价值
- 迫使你明确假设：增长率、终值、贴现率
- 关键输出：安全边际

**方法二：相对估值**
- P/E、EV/EBITDA、PEG等倍数
- 与行业同行对比
- 单独使用很危险

更全面的估值框架：
- PEG比率（P/E / 增长率）< 1.0 = 可能低估，> 2.0 = 可能高估
- EV/EBITDA — 消除资本结构差异，比P/E更公平
- P/S（市销率）— 亏损公司看这个
- P/B — 银行和资产密集型行业看这个

终极法则：当前股价隐含的增长预期 vs 公司实际增长能力，这个差才是你赚钱的空间。

### Step 4: 评估资本配置
最关键指标：ROIC vs WACC
- ROIC > WACC → 创造价值 ✓
- ROIC < WACC → 摧毁价值 ✗

资本配置决策检查清单：
- 股票回购：在高位还是低位回购？
- 收购记录：支付价格是否合理？是否创造价值？
- 股息政策：派息率是否可持续？
- 内部人持股：管理层是否有实质性利益？

### Step 5: 做决策
安全边际：
- 高质量、可预测现金流 → 20-25%
- 周期性或较低质量 → 35-50%

买入前必查清单（12项）：
1. 阅读最新10-K及至少两个季度10-Q
2. 确认经营现金流持续超过净利润（现金转换>1.0x）
3. 检查GAAP与调整后盈利差距（超过20%需警惕）
4. 计算ROIC并确认超过WACC
5. 运行DCF模型并记录关键假设
6. 运行敏感性分析（贴现率+200个基点是否仍成立）
7. 比较当前P/E、EV/EBITDA vs 5年均值及行业同行
8. 审查管理层过去5年资本配置记录
9. 检查内部人持股和近期内部人交易
10. 识别前3大风险并写下止损条件
11. 确认安全边际达标
12. 仓位大小与确信度匹配

## 二、技术面分析 3 步法 + 指标详解

### 核心框架
1. **判断趋势** — 价格在200日均线之上=牛市，之下=熊市。只在趋势方向上交易。
2. **找入场点** — 支撑压力位+K线形态+确认信号，三个同时出现才动手
3. **设止损** — 进场前就定好，放在关键支撑/压力下方1-2%

### 均线 (Moving Averages)
- SMA vs EMA：简单均线看大趋势，指数均线反应更灵敏
- 20日EMA：短线交易者的生命线
- 50日SMA：机构关注线，跌破常引发卖出
- 200日SMA：牛熊分界线
- 黄金交叉(50上穿200) = 看涨；死亡交叉(50下穿200) = 看跌
- 实战法则：只在价格位于200日均线之上时做多

### RSI (相对强弱指标)
- 超买/超卖的误区：高于70不等于立刻卖出，在强趋势中RSI可以长期停留在70以上
- **背离才是RSI最准的信号**：
  - 看涨背离：股价新低但RSI没新低 → 卖压衰竭
  - 看跌背离：股价新高但RSI没新高 → 买力衰退
- 趋势中RSI区域调整：上升趋势RSI回调到40-50是买点；下降趋势RSI反弹到50-60是卖点
- 常见错误：RSI到80就慌着卖——在强趋势中这是最贵的错误

### MACD (趋势+动量确认)
- 三大组件：MACD线(12EMA-26EMA)、信号线(9日EMA)、柱状图(差值)
- 金叉/死叉的位置很关键：零轴上方的金叉比零轴下方的强得多
- 柱状图收缩 → 早于交叉的预警信号
- **最值钱的信号：背离** — 价格创新高但MACD高点下移 = 上涨动能在衰竭

### 布林带
- 开口收缩 → 大级别变盘在即
- 沿轨运行 → 趋势强劲的标志（不是反转信号）
- 触轨非信号 → 必须由其他指标确认

### 综合运用法则
- 趋势市用均线+MACD；震荡市用RSI+布林带
- 3-5个互补指标足矣，太多导致"分析瘫痪"
- 指标提供概率优势，非绝对预测，必须结合止损

## 三、投资心理学

### 三大情绪陷阱
1. **恐惧** — 跌了就恐慌卖出，割在最低点
2. **贪婪** — 赚了还想更多，坐过山车
3. **FOMO** — 看别人赚钱就冲进去，买在山顶

### 损失厌恶
核心发现：亏损100元的痛苦 ≈ 赚200元的快乐。这就是为什么大多数人"输钱死扛、赢钱就跑"。

解决方法：
- 进场前设止损
- 仓位控制（每笔最多亏1-2%总资金）
- 止损不是认输，是付保费

### 5种最害人的认知偏差
1. **确认偏差** — 只看支持自己观点的信息
2. **锚定效应** — 被第一个数字锚住（"我100块买的，等回本再卖"）
3. **沉没成本** — "已经亏了这么多，不能走"
4. **从众效应** — "大家都在买" → 买在山顶
5. **过度自信** — "这次我看得准" → 重仓押注

### 恐惧贪婪指数
- 0-25 = 极度恐惧 → 历史上最好的买入时机
- 75-100 = 极度贪婪 → 历史上最好的减仓时机

## 四、美股市场文化

### 交易员俚语（2025 Nasdaq版）
**持仓文化：**
- Diamond Hands = 死拿不卖
- Paper Hands = 稍有波动就跑
- HODL = 无论如何都持有

**行情描述：**
- Mooning = 暴涨（"To the Moon!"）
- BTFD = Buy The F***ing Dip（抄底）
- Chop = 横盘震荡
- Bull Trap = 假突破诱多
- Toilet Bowl = 螺旋下跌

**社区语言：**
- Apes = 散户自称（来自Reddit r/WallStreetBets）
- DD = Due Diligence（尽职调查/研究）
- FUD = Fear, Uncertainty, Doubt（散布恐慌）
- Tendies = 利润/盈利
- Bagholder = 接盘侠
- Whale = 大户

**交易风格：**
- Scalping = 超短线（分钟级）
- Swing = 波段（天到周）
- YOLO = 全押

### 散户 vs 机构
散户（20-25%交易量）：追涨杀跌，FOMO驱动，持仓短
机构（75-80%交易量）：算法交易，信息优势，持仓长

散户怎么赢：速度优势、小盘股深度研究、长期持有、逆向操作

### 交易风格选择
- 日内交易：全职盯盘，90%亏钱
- 波段交易：最适合大多数人
- 长期投资：最稳，巴菲特风格
- 定投DCA：最省心，年化8-12%

### 市场开发策略
1. 教育驱动 — 先教后卖，转化率比硬推高3-5倍
2. 差异化定位 — 说出具体的，不说"服务全面"
3. 社交证明 — 案例和数据比口号有效100倍

客户类型：
- 新手（60%+）：怕亏、需要手把手
- 有经验者（30%）：要工具、要效率
- 机构/高净值（10%）：要定制、要结果
